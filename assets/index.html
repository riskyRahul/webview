Intents

You can add a special link to your HTML file and you will be able to run appropriate external app to perform some action. For example send an email or call somebody. Following intents are supported.

E-mail:

mailto:youremail@example.com?subject=Hello

Phone call:

tel:+0123456789

SMS:

sms:+0123456789

Map point:

geo:22.3113907,70.7763053

Map search:

geo:0,0?q=India

Links

URL links can be opened in internal webview or in external browser. you can easily customize rules for opening links.

For example following link is opened internally

https://android.com/?target=webview

For example following link is opened externally

https://android.com/?target=blank
